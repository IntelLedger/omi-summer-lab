Class: StatusChecker

omi-client~StatusChecker

A StatusChecker object provides a function to query for the current status of a submitted transaction. That is, whether or not the transaction has been committed to the block chain.

new StatusChecker(batchId, statusUrl)

Methods

check(timeoutopt) → {Promise.<string>}

Returns the batch status from a transaction submission. The status is one of ['PENDING', 'COMMITTED', 'INVALID', 'UNKNOWN'].

The status is returned via a promise. If a timeout occurs, the promise is rejected.
